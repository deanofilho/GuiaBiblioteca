# Guia da Biblioteca

Este aplicativo Android foi desenvolvido como um **Projeto de Extensão Universitária**. 
O objetivo é facilitar o acesso da comunidade acadêmica aos serviços da Biblioteca.

## 🚀 Funcionalidades
- **Consulta ao Acervo:** Integração otimizada com o portal de busca da UFNT.
- **FAQ Inteligente:** Guia completo de regras, multas e serviços com interface intuitiva.
- **Comunicação Direta:** Atalhos para WhatsApp, E-mail e localização via Google Maps.
- **Acessibilidade:** Textos estruturados para melhor leitura e navegação simplificada.

## 🛠️ Tecnologias Utilizadas
- Java & Android SDK
- WebView com injeção de CSS/JavaScript para melhoria de UX
- OnBackPressedDispatcher (Navegação moderna Android 13+)
- Material Design Components
